## paygentic entitlements grants get

Get Grant

### Synopsis

Retrieve a specific grant by ID.

```
paygentic entitlements grants get [flags]
```

### Examples

```
  paygentic grants get --entitlement-id <id> --grant-id <id>
```

### Options

```
  -e, --entitlement-id string   The unique identifier of the entitlement. [required]
  -g, --grant-id string         The unique identifier of the grant. [required]
  -h, --help                    help for get
```
